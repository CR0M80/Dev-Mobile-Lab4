# 📋 FragmentsLab — By SAAD

Application Android simple permettant de découvrir les **fragments dynamiques**, la navigation et les interactions utilisateur.

---

## 🎯 Objectif

* Comprendre les fragments
* Naviguer entre plusieurs interfaces
* Gérer les événements utilisateur
* Observer le cycle de vie

---

## 🖼️ Interface

L’application contient :

* 🔘 Deux boutons pour naviguer entre les fragments
* 🧩 Un conteneur dynamique pour afficher les fragments

### Fragment 1

* Affiche un texte
* Bouton interactif qui modifie le contenu

### Fragment 2

* Barre de progression (SeekBar)
* Affichage dynamique de la valeur
* Changement de couleur selon la valeur

---

## ⚙️ Fonctionnement

* Au démarrage → Fragment 1 s’affiche
* Clic sur un bouton → changement de fragment
* Bouton "Retour" → navigation arrière
* Les interactions mettent à jour l’interface en temps réel

---

## 🧠 Concepts utilisés

* Fragment
* FragmentManager
* Navigation dynamique
* Gestion des événements
* Cycle de vie
* Sauvegarde d’état

---

## 🚀 Résultat

Une application simple mais complète pour maîtriser les bases des fragments Android avec une interface moderne et interactive.
